## **What This Is**

I made a simple chatbot that can answer questions about Python programming. It's like a study buddy that knows about basic coding concepts. You can ask it things like:
- "what's a variable?"
- "how do functions work?" 
- "tell me about loops"

It's not super smart like the AI in movies, but it actually works! I built it using only the stuff I've learned in my first semester.

## **How I Made It**

I used these things from my classes:
- **Dictionaries** to store all the answers
- **If/else statements** to make decisions
- **While loops** to keep the conversation going
- **Functions** to organize my code (this was hard but I got it!)
- **String methods** to understand what people type
